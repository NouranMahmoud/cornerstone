This is a sample of what a proper paginator does. It can only be used in one directory on the site, but is able to access all posts from all categories.

{% for post in paginator.posts %}

{{ post.title }}

Published on {{ post.date | date:"%B %d, %Y" }} {% if post.author %} by {{ post.author }}{% endif %}

{{ post.excerpt }}
{% endfor %}
{% if paginator.previous_page %} Previous page {% else %} Previous page {% endif %} Page: {{ paginator.page }} of {{ paginator.total_pages }} pages {% if paginator.next_page %} Next page {% else %} Next page {% endif %}
